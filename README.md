# 🎬 React Movie App

## 📌 Description
This is a React-based movie application where users can browse and explore movies with a clean and responsive UI.

## 🚀 Features
- Browse movies
- View movie details
- Responsive design
- Clean UI with SCSS styling

## 🛠️ Tech Stack
- React.js
- JavaScript
- SCSS
- HTML5

## ▶️ Getting Started

### Install dependencies
npm install

### Run the app
npm start

## 📂 Project Structure
- src/ → main source code
- public/ → static files
